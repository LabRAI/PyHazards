How to Cite
===========

Use the following citation for the PyHazards software package itself. If you are
also relying on specific datasets or model papers, cite those sources from
:doc:`references` as well.

Library Citation
----------------

.. code-block:: bibtex

   @misc{pyhazards2025,
     title        = {PyHazards: An Open-Source Library for AI-Powered Hazard Prediction},
     author       = {Cheng et al.},
     year         = {2025},
     howpublished = {\url{https://github.com/LabRAI/PyHazards}},
     note         = {GitHub repository}
   }
