Wildfire FPA Forecast
=====================

Description
-----------

``wildfire_fpa_forecast`` exposes the temporal forecasting stage of the broader FPA-FOD wildfire framework as its own registry entry.


The PyHazards adapter keeps the paper-inspired forecasting path while using the shared builder contract and smoke-test interface.


Paper
-----

See `Developing risk assessment framework for wildfire in the United States <https://www.sciencedirect.com/science/article/pii/S2949926723000033>`_.

Registry Name
-------------

Primary entrypoint: ``wildfire_fpa_forecast``

Supported Tasks
---------------

- ``regression``
- ``forecasting``

Example of how to use it
------------------------

.. code-block:: python

   import torch
   from pyhazards.models import build_model

   model = build_model(
       name="wildfire_fpa_forecast",
       task="forecasting",
       input_dim=5,
       output_dim=5,
       lookback=12,
   )
   preds = model(torch.randn(2, 12, 5))
   print(preds.shape)

Notes
