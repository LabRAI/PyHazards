Wildfire FPA LSTM
=================

Description
-----------

``wildfire_fpa_lstm`` is the lighter-weight temporal baseline in the wildfire FPA family and complements the autoencoder-enhanced forecast model.


It preserves the shared lookback-window regression contract used across the PyHazards weekly wildfire smoke tests.


Paper
-----

See `Developing risk assessment framework for wildfire in the United States <https://www.sciencedirect.com/science/article/pii/S2949926723000033>`_.

Registry Name
-------------

Primary entrypoint: ``wildfire_fpa_lstm``

Supported Tasks
---------------

- ``regression``
- ``forecasting``

Example of how to use it
------------------------

.. code-block:: python

   import torch
   from pyhazards.models import build_model

   model = build_model(
       name="wildfire_fpa_lstm",
       task="forecasting",
       input_dim=5,
       output_dim=5,
       lookback=12,
   )
   preds = model(torch.randn(2, 12, 5))
   print(preds.shape)
