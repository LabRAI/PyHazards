DNN-LSTM-AutoEncoder
====================

Description
-----------

``wildfire_fpa`` is the paper-facing PyHazards entrypoint for the FPA-FOD wildfire framework described by Shen et al. (2023).


PyHazards exposes the classification and forecasting stages through one public registry name while keeping the lower-level modules available internally.


Paper
-----

See `Developing risk assessment framework for wildfire in the United States <https://www.sciencedirect.com/science/article/pii/S2949926723000033>`_.

Registry Name
-------------

Primary entrypoint: ``wildfire_fpa``

Supported Tasks
---------------

- ``classification``
- ``forecasting``
- ``regression``

Example of how to use it
------------------------

.. code-block:: python

   import torch
   from pyhazards.models import build_model

   model = build_model(
       name="wildfire_fpa",
       task="classification",
       in_dim=8,
       out_dim=5,
       hidden_dim=64,
       depth=2,
   )

   x = torch.randn(4, 8)
   logits = model(x)
   print(logits.shape)

Notes
-----

- Use ``task="classification"`` for the DNN stage.
- Use ``task="forecasting"`` or ``task="regression"`` for the sequence stage.
