Wildfire FPA DNN
================

Description
-----------

``wildfire_fpa_dnn`` exposes the classification stage of the broader ``wildfire_fpa`` pipeline as a standalone public baseline.


It is useful when contributors want a simple wildfire danger benchmark without the sequence-forecasting components.


Paper
-----

See `Developing risk assessment framework for wildfire in the United States <https://www.sciencedirect.com/science/article/pii/S2949926723000033>`_.

Registry Name
-------------

Primary entrypoint: ``wildfire_fpa_dnn``

Supported Tasks
---------------

- ``classification``

Example of how to use it
------------------------

   import torch
   from pyhazards.models import build_model

   model = build_model(name="wildfire_fpa_dnn", task="classification", in_dim=8, out_dim=5)
